PreviewLite
===========

「Windows 画像と FAX ビューア」のコンポーネントとして提供されていた `Preview Class` の模造品を提供します。

画像処理には [CxImage](http://www.codeproject.com/Articles/1300/CxImage) を利用しています。

対応: `bmp` `jpg` `png` `tif` `wbmp` `wmf`

↓ このような場合、PreviewLite が代替になるかもしれません。
